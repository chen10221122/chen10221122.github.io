3-4、什么是和为什么要使用 ReactHook

  • React 16.8 带来的全新特性，即将替代 class 组件的写法
  • 没有破坏性改动
    1. 完全可选，无需重写已有代码
    2. 百分之百向后兼容
    3. 没有计划从 React 移除 class

函数组件有局限，在组件内不能使用 state 和生命周期的一些钩子函数， Hock 是用函数式组件代替 class 组件的一种写法

Hook 解决的问题

  • 组件很难复用状态逻辑

  • 复杂组件难以理解，尤其是生命周期函数
  • React 组件一直是函数，使用 Hook 完全拥抱函数
