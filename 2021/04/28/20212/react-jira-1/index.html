JSX 被编译成什么

  • JSX 格式是一种语法糖 - 替代 React.createElement() 方法
  • 方法返回 ReactElement对象

console.log 之后输出一个有一堆属性的 Object，复杂的 JSX 也不过是 Object 的嵌套而已，这些 Object 被称为 ReactElement.

ReactDOM.render 方法就是把这些 ReactElement 一些复杂的 Object，通过算法渲染到真实的 DOM 节点上，最后呈现到页面上。
