6、小程序弹窗 使用 catchtouchmove 的问题

使用 fixed 定位之后，滑动弹出的视图，发现根页面 page 也跟着滑动，事件会穿透

解决：

  1. 弹窗背景使用 fixed 定位，占满全屏，设置半透明

  2. 设置弹窗最上层 view 的属性 catchtouchmove=“方法名”，这里的方法完全是为了消除控制台的警告，可不做处理。

  3. 在弹窗里面具体显示的内容使用 scroll-view，并注意设置 scroll-view 的高度

  4. 再给 scroll-view 设置属性 scroll-y，就可以了。(ps：切记一定要设置 scroll-y，否则将不会起任何效果，导致 scroll-view 不能滑动)
