七、微信小程序自定义头部以及页面高度计算

!!! 想让每个页面都有默认头部，则app.js中config一定不要配置 navigationStyle: ‘custom’

  • 在指定页面自定义头部时，在该页面的config中配置navigationStyle: ‘custom’，并封装头部组件

  • 如果设置了navigationStyle: ‘custom’，则屏幕高度 == 可使用窗口高度 == screenHeight

  • windowHeight为可使用窗口高度，screenHeight为屏幕高度

十一、小程序windowHeight在不同页面高度不一致问题及解决方案

关于小程序各个高度的新理解：
iphone6为例：
screenHeight：窗口满屏高度667
statusBarHeight：状态栏高度20
windowHeight：分为两种情况
当有导航栏时，导航栏高度是44，windowHeight等于603
当没有导航栏时，windowHeight等于667

注意：计算高度时量取视觉稿（2倍）的高度和宽度都要除以2

如果设置了navigationStyle: ‘custom’，则屏幕高度 == 可使用窗口高度 == screenHeight

  • windowHeight 概念

可使用窗口高度，即：屏幕高度（screenHeight） - 导航（tabbar）高度 - 导航栏高度 - 状态栏高度。

在不含导航（tabbar）的页面，windowHeight =屏幕高度（screenHeight） - 导航栏高度 - 状态栏高度

  • 使用windowHeight

在页面中调用wx.getSystemInfo即可获取当前页面的屏幕高度，
