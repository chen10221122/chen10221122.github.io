javascript拖拽事件
更新于 2021-04-23 javascript

需求：点击 div 拖拽到指定位置，鼠标松开后回到原位。

一、clientX、clientY
点击位置距离当前 body 可视区域的 x，y 坐标

二、pageX、pageY
对于整个页面来说，包括了被卷去的 body 部分的长度

三、screenX、screenY
点击位置距离当前电脑屏幕的 x，y 坐标

四、offsetX、offsetY
相对于带有定位的父盒子的 x，y 坐标

五、x、y
和 screenX、screenY 一样
