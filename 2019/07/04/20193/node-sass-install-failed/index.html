node-sass安装失败的n中解决方法
更新于 2021-04-25 angular

node-sass 安装时有时会出现各种问题，本文介绍我自己遇到的问题和可行的解决方法。

解决方案 1、切换淘宝镜像

1 卸载 node-sass

2 安装 node-sass

上一句失败 将 已下载东西 重新编译

3 安装其余的依赖

解决方案 2、使用本地的 node-sass 包

官方 git 版本地址：https://github.com/sass/node-sass/releases

注:下载的 node-sass 要改名。同时注意 node-sass 的版本。

其他：相似方案 1,相似方案 2

解决方案 3、更换其他版本 node.js

观察报错信息是否提到了 nodejs 版本的问题，如果是的话可以替换掉支持的版本。

我从控制面板-程序中卸载 node.js。下载 v8.1.6 的版本。

https://nodejs.org/dist/latest-v8.x/

解决方法 4、更换其他版本 node-sass

angular 更新到了 8 之后，提示 nodejs8.x 已经不支持了，所以升到了 10.x;

这个时候 node-sass 又不行了：

解决方法：
去 node-sass 的 git 仓库https://github.com/sass/node-sass找 node 对应的 node-sass 版本

然后在项目的package.json里修改对应的版本号

之后删除node-sass并重新安装

**删除全部node_modules
