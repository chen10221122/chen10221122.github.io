  • 问题原因：处理动画等情况下必须操作 dom

4.2 父子组件的数据传递

父组件通过组件的形式向子组件传值

子组件通过事件触发形式向父组件传值

单向数据流：子组件不能修改父组件传递的数据

4.3 组件参数校验与非 Props 特性

参数校验：类型，必填，默认，校验

Props 特性：

  • 父组件给子组件传值有对应关系（比如说父组件传递了hello,子组件再props里接收了hello）
  • 特点：属性（如：content)不会在DOM标签中显示；子组件可以用this.content调用

非 Props 特性：

  • 子组件并没有声明props这个对象,故无法获取父组件内容；
  • 属性会显示在 DOM 标签

4.4 给组件绑定原生事件

一般方法：

  • 1、子组件里面用@click绑定事件；
  • 2、子组件用this.$emit('click')触发父组件事件；

原生事件：native修饰符

  • 使用click.native监听原生事件，可以直接在父组件使用

4.5 非父子组件间的传值

假设第一层组件与第三层组件传值。或者第三层组件间传值。

Vue 是一个轻量化的组件，无法完成复杂的传值，可以借助其他方法：

  • 1、借助 Vue 官方提供的数据层的框架：vuex

  • 2、使用发布订阅模式(Bus/总线/发布订阅模式/观察者模式)

五、 Vue 中的动画特效

5.1 Vue 中 CSS 动画原理

利用 css3 实现，又称过渡动画效果

  • 元素显示：

fade-enter：动画第一帧执行，第二帧移除

fade-enter-active：动画第一帧执行，结束后移除

fade-enter-to：动画第二帧执行，结束后移除

  • 元素隐藏：

fade-leave：动画第一帧执行，第二帧移除

fade-leave-active：动画第一帧执行，结束后移除

fade-leave-to：动画第二帧执行，结束后移除

5.2 在 vue 中使用 animate.css 库

  • 自定义动画使用：@keyframes

  • 不使用name绑定的样式名称，使用自定义 class 名：可以使用animate.css库。该库使用@keyframes

  • class 类里面必须包含animated类名

5.3 Vue 中同时使用过渡和动画

  • 页面首次加载动画：使用appear和appear-active-class

  • 入场动画同时有@keyframe动画和过渡动画：属性添加多个参数

  • 两个动画时动画时长：type="transition"

  • 手动设置动画时长：:duration

5.4 Vue 中的 Js 动画与 Velocity.js 的结合

  • 通过 js 钩子实现：

入场动画：@before-enter、@enter、@after-enter

出场动画：@before-leave、@leave、@after-leave
