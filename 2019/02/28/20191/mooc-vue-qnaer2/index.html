6.4 单页应用 VS 多页应用

多页面应用：

  • 优点：首屏时间快，SEO（搜索引擎优化）效果好
  • 缺点：页面切换满
    单页面应用：
    通过 js 感知 Url 的变化，动态的清除当前组件，渲染新的组件
  • 优点：页面切换快
  • 缺点：首屏时间稍慢，SEO 差
  • 解决方法：服务器端渲染
